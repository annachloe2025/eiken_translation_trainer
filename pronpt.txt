指示内容
以下に、日本語の文章とその英訳を送ります。これを添削し、翻訳スキル向上の助けとなるフィードバックを以下のフォーマットで出力してください。

タイトル

翻訳に関連するタイトルを記載してください。
原文の日本語

日本語の原文をそのまま記載してください。
ユーザーによる英訳

ユーザーが翻訳した英訳をそのまま記載してください。
修正後の英訳

添削後の正しい英訳を記載してください。
エラー一覧

ユーザーの英訳での間違いをすべてリストアップし、それぞれの理由を日本語で説明してください。
翻訳の採点と解説

ユーザーの英訳を100点満点で採点し、そのスコアの理由を具体的に解説してください。
改善アドバイス

翻訳スキルを向上させるための具体的なアドバイスを提供してください。
覚えておいた方が良いフレーズや単語

翻訳に関連して知っておくと役立つ単語やフレーズをリストアップし、それぞれの意味や使い方を簡単に説明してください。
フォーマット
以下のJSONフォーマットで出力してください：

{
  "title": "タイトル",
  "original_japanese": "ここに日本語の文章",
  "user_translation": "ここにユーザーの英訳",
  "corrected_translation": "ここに修正された英訳",
  "errors": [
    {
      "error": "エラーの説明",
      "reason": "日本語でエラーの理由"
    }
  ],
  "score": {
    "value": 0,
    "explanation": "スコアの説明"
  },
  "suggestions": [
    "改善のアドバイス",
    "必要ならば複数"
  ],
  "words": [
    {
      "word": "知っておくとよい単語",
      "definition": "単語の意味"
    },
    {
      "word": "",
      "definition": ""
    }
  ]
}
注意点
エラー一覧は、誤った文法、語彙の選択ミス、文脈の不一致など、すべての問題点を詳しく記載してください。
修正後の英訳は、ユーザーの意図を正確に反映し、自然な英語表現にしてください。
採点の理由では、評価基準（例：正確さ、流暢さ、語彙選択など）を明確に示してください。
改善アドバイスでは、具体的な練習方法や参考資料を含めると効果的です。
覚えておくべき単語は、実用的で、翻訳に役立つものを優先してください。
